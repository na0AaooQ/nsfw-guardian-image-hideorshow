1. Introduction

“Gazou Mimamori | Sensitive image filter for X” (the “Extension”) is a Chrome extension that automatically detects and blocks uncomfortable or explicit images displayed in your X (formerly Twitter) timeline and direct messages.

This page explains what information the Extension handles and how that information is treated.

For an overview of the Extension, please see the 📖 Gazou Mimamori overview PDF.

For instructions on how to use the Extension, please see the 📖 User Manual.

2. Information We Collect

The Extension collects and processes the following information.

Type of information Purpose External transmission
Image data displayed on X To determine whether an image is uncomfortable or explicit, processed only inside your browser None
Extension settings
(enabled/disabled status and sensitivity threshold)		 Saved in Chrome local synced storage None

3. Information We Do Not Collect

The Extension does not collect, record, or transmit the following information.

  • Information that can identify you personally, such as your name, email address, or home address
  • Your X account information or login credentials
  • Your browsing history, post content, or direct message content
  • The image data itself after judgment
  • Logs of judgment results or scores

4. How Information Is Processed

All processing by the Extension is completed inside your browser (Chrome).

  • Image judgment runs in the browser using the Offscreen Document API
  • The AI model (NSFWJS / TensorFlow.js) is loaded in the browser and performs inference locally
  • Judgment results are used only to control what is displayed on the screen, and are not saved or transmitted
  • Settings (enabled/disabled status and sensitivity) are saved in Chrome storage.sync

5. Disclosure to Third Parties

The Extension does not provide, sell, or share information with any third party.

6. Permissions Used

The Chrome permissions requested by the Extension and the reasons for using them are as follows.

Permission Reason
storage To save enabled/disabled status and sensitivity settings in the browser
offscreen To run AI image judgment in the background
Host access
(x.com / twitter.com / pbs.twimg.com)		 To retrieve and judge images displayed on X

7. Security

The Extension complies with Chrome Extension Manifest V3. Its Content Security Policy (CSP) prohibits eval and the execution of external scripts. All code is publicly available in the repository for review.

  • Source code: GitHub repository
