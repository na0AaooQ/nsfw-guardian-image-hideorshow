🛡️ がぞうみまもり | Xセンシティブ画像フィルター

プライバシーポリシー

最終更新日：2026年3月15日

📌 本Google Chrome拡張機能は、あなたの画像データや個人情報を外部サーバーへ一切送信しません。すべての処理はお使いのブラウザ内のみで完結します。

1. はじめに

「がぞうみまもり | Xセンシティブ画像フィルター」（以下「本拡張機能」）は、X（旧Twitter）のタイムラインおよびDMに表示されるセンシティブな画像を自動的に検出・ブロックするChrome拡張機能です。

本ページでは、本拡張機能が取り扱う情報の内容とその扱い方について説明します。

2. 収集する情報

本拡張機能が収集・処理する情報は以下の通りです。

情報の種類 用途 外部送信
X上に表示された画像データ センシティブ画像の判定（ブラウザ内のみで処理） なし
拡張機能の設定値
（有効/無効・感度の閾値）		 Chromeのローカルストレージに保存（同期） なし

3. 収集しない情報

本拡張機能は以下の情報を収集・記録・送信しません。

  • 氏名・メールアドレス・住所等の個人を特定できる情報
  • Xのアカウント情報・ログイン情報
  • 閲覧履歴・投稿内容・DMの内容
  • 判定した画像データそのもの
  • 判定結果・スコアのログ

4. 情報の処理方法

本拡張機能のすべての処理はお使いのブラウザ（Chrome）内で完結します。

  • 画像判定はブラウザ内の Offscreen Document API 上で実行されます
  • AIモデル（NSFWJS / TensorFlow.js）はブラウザ内にロードされ、ローカルで推論を行います
  • 判定結果は画面表示の制御のみに使用され、保存・送信されません
  • 設定値（有効/無効・感度）はChromeの storage.sync に保存されます

5. 第三者への情報提供

本拡張機能は、いかなる第三者に対しても情報の提供・販売・共有を行いません。

6. 利用する権限について

本拡張機能が要求するChrome権限とその理由は以下の通りです。

権限 理由
storage 有効/無効・感度設定をブラウザに保存するため
offscreen バックグラウンドでAI画像判定を実行するため
ホストアクセス
（x.com / twitter.com / pbs.twimg.com）		 X上の画像を取得・判定するため

7. セキュリティ

本拡張機能は Chrome拡張機能 Manifest V3 に準拠しており、Content Security Policy（CSP）により eval や外部スクリプトの実行が禁止されています。すべてのコードはリポジトリ上で公開されており、誰でも確認できます。

  • ソースコード：GitHub リポジトリ

8. お子様のプライバシー

本拡張機能は13歳未満のお子様を対象としたサービスではありません。13歳未満のお子様の個人情報を意図的に収集することはありません。

9. プライバシーポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は、本ページ上部の「最終更新日」を更新します。重要な変更がある場合は、リポジトリの README にも記載します。
